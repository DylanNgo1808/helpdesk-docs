# Templates

Design invoices that reflect your brand. This section shows you how to browse available templates, create your own, and fine-tune every detail from layout and colors to content fields and custom CSS.

## Guides

- [Browse Templates](/docs/templates/browse-templates) — Explore the built-in template library
- [Create a Template](/docs/templates/create-template) — Start a new template from scratch
- [Edit General Settings](/docs/templates/edit-general) — Adjust template name, format, and basic options
- [Customize Theme](/docs/templates/edit-theme) — Change colors, fonts, and visual styling
- [Edit Content & Fields](/docs/templates/edit-content) — Choose which fields and sections appear on invoices
- [Edit Design & CSS](/docs/templates/edit-design) — Apply custom CSS for advanced layout control
- [Preview & Test](/docs/templates/preview-template) — Preview your template before using it
- [Manage Templates](/docs/templates/manage-templates) — Duplicate, rename, delete, or set a default template

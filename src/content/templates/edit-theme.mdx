# How to Customize Template Theme

The **Theme** tab lets you select the base design layout for your template and set it as the default print template. Each theme provides a distinct visual style for your PDF documents.

## Prerequisites

- Avada PDF Invoice app installed on your Shopify store
- An existing template open in the editor
- Access to the app admin dashboard

## Steps

### Step 1: Open the template editor

From the **Templates** page, click the **Edit** button on the template you want to customize.

{/* screenshot: Templates page with Edit button highlighted on a template card */}

### Step 2: Navigate to the Theme tab

In the left sidebar, click the **Theme** menu item. The help text reads "Edit details inside the template." The right panel displays the theme selection options.

{/* screenshot: Left sidebar with Theme tab selected */}

### Step 3: Set as default print template

At the top of the Theme panel, check or uncheck the **Default** checkbox. When enabled, this template becomes the default for printing, sending, and downloading documents of its type. The help text explains: "Check to make this template default for print, send, and download. Only one template of each document type can be applied."

{/* screenshot: Default checkbox at the top of the Theme panel */}

### Step 4: Review the default template warning

If another template is already set as the default for the same document type, an info banner appears: "A default print template already exists. If you choose this one, it will replace the current default." This lets you know that selecting this template as default will unset the previous one.

{/* screenshot: Info banner about replacing the current default template */}

### Step 5: Browse available template designs

Scroll through the list of template designs below the default checkbox. Each template shows:

- A **thumbnail preview** of the design layout
- The **template name** (e.g., "Spring", "Elegant")
- A **Select** button to choose the design
- A **plan badge** (Professional, Ultimate) if the template requires a paid plan

Templates are filtered to match the current document type (Invoice, Packing slip, Refund, etc.).

{/* screenshot: List of template designs with thumbnails and Select buttons */}

### Step 6: Select a template design

Click the **Select** button on the template design you want to use. A confirmation modal opens showing the full template details and a larger preview. Click to confirm your selection. The template layout, colors, and fonts automatically update to match the chosen design.

{/* screenshot: Template selection confirmation modal */}

### Step 7: Upgrade for premium templates

Templates that require a higher plan display a **Professional** or **Ultimate** badge. Clicking **Select** on a premium template redirects you to the Subscription page to upgrade your plan.

{/* screenshot: Premium template with "Professional" badge and locked Select button */}

### Step 8: Unlock a free premium template

If you are not on a paid plan, a banner appears at the top: "Unlock one more premium template from paid plans. Contact us now to get it free!" Click **Get template** to contact the support team for a complimentary premium template.

{/* screenshot: Banner offering a free premium template with "Get template" button */}

### Step 9: Preview the changes

After selecting a new template design, the center preview panel refreshes to show the updated layout. Review the preview to make sure the design matches your expectations.

{/* screenshot: Updated live preview after selecting a new template design */}

### Step 10: Save your changes

Click **Save** in the save bar at the bottom of the screen to apply the new theme. A success toast notification confirms the changes were saved.

{/* screenshot: Save bar with Save button highlighted */}

## Tips

- Only one template per document type can be the default -- selecting a new default automatically replaces the previous one
- Premium templates offer more polished and varied designs. Contact support to get one free premium template unlocked
- Changing the template theme may reset certain design settings (like primary color and font). Review the Design tab after switching themes
- Template designs are categorized by document type, so you will only see themes that match your current document type selection

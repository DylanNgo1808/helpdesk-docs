# How to Manage Templates

The **Templates** page provides tools to organize your template library. You can set a default print template, edit existing templates, delete templates you no longer need, and select or unselect templates for use. This article covers the key management actions available from the Templates page and template editor.

## Prerequisites

- Better PDF Invoice app installed on your Shopify store
- At least one template created
- Access to the app admin dashboard

## Steps

### Step 1: Open the Templates page

From the Better PDF Invoice sidebar menu, click **Templates** to view all your templates in a grid layout.

{/* screenshot: Templates page showing the grid of template cards */}

### Step 2: Set a template as the default

To set a template as the default for printing, sending, and downloading, you have two options:

**From the Templates page:** Click the **Select** button on the template card. The template becomes the default for its document type.

**From the template editor:** Open the template, go to the **Theme** tab, and check the **Default** checkbox. The help text reads: "Check to make this template default for print, send, and download. Only one template of each document type can be applied."

{/* screenshot: Template card with the Select button, and Theme tab with Default checkbox */}

### Step 3: Unselect a default template

To remove a template as the default, click the **Unselect** button on the template card (which appears instead of **Select** when the template is currently the default). Alternatively, open the template editor, go to the **Theme** tab, and uncheck the **Default** checkbox.

{/* screenshot: Template card showing the Unselect button */}

### Step 4: Edit an existing template

Click the **Edit** button on a template card to open the full-screen template editor. From there, you can modify all settings across the four tabs:

- **General** -- Template name, currency, order type, document type
- **Theme** -- Base template design, default print setting
- **Content** -- Data fields and toggles
- **Design** -- Visual settings (logo, font, color, CSS)

Make your changes, then click **Save** to apply.

{/* screenshot: Template editor showing the four sidebar tabs */}

### Step 5: Delete a single template

Click the **Delete** button on the template card you want to remove. A confirmation modal appears with the title **Permanent Deletion Warning** and the message: "This template may be used for emails. Deleting is permanent."

Click **Delete anyway** to confirm the deletion, or click **Cancel** to keep the template.

{/* screenshot: Delete confirmation modal with "Delete anyway" and "Cancel" buttons */}

### Step 6: Delete from the preview modal

You can also delete a template from the preview modal. Click **Preview** on a template card, then in the modal, click the **Delete** button in the footer. The same confirmation dialog appears.

{/* screenshot: Preview modal with the Delete button in the footer */}

### Step 7: Edit from the preview modal

In the preview modal, click the **Edit** button to navigate directly to the template editor. This provides a quick shortcut from previewing a template to editing it.

{/* screenshot: Preview modal with the Edit button highlighted */}

### Step 8: Upload or change your store logo

In the template editor, the left sidebar shows the logo upload area under **Template settings**. Click to upload a new logo (JPEG, JPG, or PNG, maximum 1 MB). The logo is shared across templates and appears on all your PDF documents.

{/* screenshot: Logo upload area in the template editor sidebar */}

### Step 9: Discard unsaved changes

If you make changes in the template editor and want to revert them, click **Discard** in the save bar at the bottom. All settings reset to their last saved state. If you try to navigate away with unsaved changes, a leave confirmation dialog appears asking whether to save or discard.

{/* screenshot: Save bar with the Discard button, and leave confirmation dialog */}

### Step 10: Get help customizing templates

If you need professional help designing your templates, a callout banner may appear on the Templates page: "Want to customize your documents for a professional look?" Click **Contact us** to reach the support team, who can help create custom template designs for your store.

{/* screenshot: Callout card banner with "Contact us" button */}

## Tips

- Only one template per document type can be set as the default. Setting a new default automatically replaces the previous one
- Deleting a template is permanent and cannot be undone. Make sure the template is not in use for email automation before deleting
- The delete confirmation warns about email associations because templates can be linked to automated email delivery methods
- Use descriptive template names to easily identify templates when managing a large library
- If you accidentally delete a template, you will need to create a new one from scratch. Consider exporting your Liquid code (from the Liquid tab) as a backup before deleting

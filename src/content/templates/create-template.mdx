# How to Create a New Template

Creating a new template lets you design a custom PDF document layout for your invoices, packing slips, refunds, or other document types. The template editor opens in a full-screen view with a live preview, sidebar settings, and menu tabs.

## Prerequisites

- Avada PDF Invoice app installed on your Shopify store
- Access to the app admin dashboard

## Steps

### Step 1: Navigate to the Templates page

From the Avada PDF Invoice sidebar menu, click **Templates** to open the templates list.

{/* screenshot: Sidebar menu with Templates highlighted */}

### Step 2: Click "New template"

Click the **New template** button in the top-right corner of the Templates page. This opens the full-screen template editor.

{/* screenshot: Templates page with the "New template" button highlighted */}

### Step 3: Upload your store logo

In the left sidebar under **Template settings**, you will see the logo upload area. Click to upload your store logo in JPEG, JPG, or PNG format (maximum 1 MB). This logo appears on your PDF documents.

{/* screenshot: Logo upload area in the template editor sidebar */}

### Step 4: Enter a template name

Click on the **General** tab in the left sidebar menu. In the **Template name** field, enter a descriptive name for your template (e.g., "Standard Invoice" or "Holiday Packing Slip"). The default name is "Untitled template."

{/* screenshot: General tab with the Template name text field */}

### Step 5: Choose the template currency

Under **Template currency**, select one of the following:

- **Store currency** -- Uses the default currency set for your Shopify store
- **Billing currency** -- Uses the currency specified during the transaction (requires Ultimate plan)

{/* screenshot: Template currency radio button options */}

### Step 6: Select the order type

Under **Order type**, choose between:

- **Order** -- For standard Shopify orders
- **Draft order** -- For draft orders and quotes

This determines which order data is available in the template.

{/* screenshot: Order type tab selector showing Order and Draft order tabs */}

### Step 7: Select the document type

From the **Document type** dropdown, choose the type of document this template will generate:

- Invoice
- Packing slip
- Refund
- Pro forma
- Credit note

For draft orders, the document type is automatically set to **Quotes**.

{/* screenshot: Document type dropdown with options listed */}

### Step 8: Configure content settings

Click the **Content** tab in the left sidebar to configure which fields appear on the document. Toggle checkboxes for order information, payment and shipping details, item information, and extra fields. See the [Edit Content](/docs/templates/edit-content) article for detailed options.

{/* screenshot: Content tab showing checkboxes for order information fields */}

### Step 9: Customize the design

Click the **Design** tab to adjust visual settings including logo size, font family, font size, primary color, and background image. See the [Edit Design](/docs/templates/edit-design) article for details.

{/* screenshot: Design tab showing logo size slider and color picker */}

### Step 10: Preview your template

The center panel shows a live PDF preview of your template. Changes to content and design settings are automatically reflected in the preview after a brief loading period.

{/* screenshot: Center panel showing the live PDF preview of the template */}

### Step 11: Save the template

Click the **Save** button in the save bar at the bottom of the screen. A success toast notification confirms the template was created. You will be redirected back to the Templates page.

{/* screenshot: Save bar at the bottom with Save and Discard buttons */}

## Tips

- Give your template a unique, descriptive name to easily identify it later
- Upload a high-quality logo (PNG with transparent background works best) for the most professional look
- Use the **Discard** button to reset all changes back to the default values before saving
- If you navigate away with unsaved changes, a confirmation dialog will prompt you to save or discard

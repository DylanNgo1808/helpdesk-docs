# How to Apply Discount Codes

Better PDF Invoice supports discount codes that reduce your subscription cost. You can apply a discount code from the subscription page before upgrading or while on a paid plan.

## Prerequisites

- An active Better PDF Invoice installation
- A valid discount code
- The discount code feature is available for users who are not on a promotional plan

## Steps

### Step 1: Navigate to the Subscription Page

From the Better PDF Invoice dashboard, click **Subscription** in the left sidebar menu.

{/* screenshot: Left sidebar with Subscription highlighted */}

### Step 2: Scroll to the Plan Details Section

Scroll down to the bottom of the subscription page to find the **Plan details** section. Below your member information, you will see the **Discount code** field.

{/* screenshot: Plan details section with Discount code field visible */}

### Step 3: Enter Your Discount Code

Type your discount code into the **Discount code** text field. The field accepts any valid promotional code provided by Better PDF Invoice.

{/* screenshot: Discount code text field with a code typed in */}

### Step 4: Click Apply

Click the **Apply** button next to the discount code field. The button appears to the right of the text input. While the code is being validated, the button shows a loading spinner.

{/* screenshot: Apply button next to the discount code field */}

### Step 5: Verify the Discount

If the code is valid, a success toast notification reading **"Applied discount successfully"** appears. The discount code field becomes disabled (masked) and the button label changes to **Cancel**, indicating the code is active.

{/* screenshot: Success toast and masked discount code field with Cancel button */}

### Step 6: Upgrade with the Discount

With the discount code applied, navigate to the plan cards above and select a plan. The discount will be applied to your subscription charge when you approve it in Shopify.

{/* screenshot: Plan cards with discount code already applied */}

### Step 7: Remove a Discount Code (Optional)

If you want to remove an applied discount, click the **Cancel** button next to the masked discount code field. A toast notification confirms the discount was cancelled with the message **"Cancelled discount successfully"**. The field becomes editable again so you can enter a different code.

{/* screenshot: Cancel button on applied discount and success toast after removal */}

## Tips

- If you enter an invalid code, the error message **"Invalid coupon code"** appears below the text field. Double-check the code for typos.
- The discount code field is only visible when you are not on a promotional plan. If you are already on a promotional offer, the field will not appear.
- Discounts are applied to Shopify billing. The reduced amount is reflected in your Shopify admin under the app subscription charges.
- You can only have one discount code active at a time. Remove the current code before applying a new one.
- Keep your discount code private. Each code may have usage limits or expiration dates.

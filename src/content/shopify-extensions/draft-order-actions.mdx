# How to Use Extensions on Draft Order Pages

Avada PDF Invoice adds action links to both the draft order detail page and the draft order list page in Shopify Admin. You can print or download a single draft order invoice, or use bulk actions to process multiple draft orders at once.

## Prerequisites

- Avada PDF Invoice app installed and active on your Shopify store
- At least one draft order created in your Shopify store
- An active subscription plan that supports printing

## Steps

### Step 1: Open a Draft Order in Shopify Admin

Navigate to **Orders > Drafts** in your Shopify Admin sidebar. Click on any draft order to open its detail page.

{/* screenshot: Shopify Admin sidebar showing Orders > Drafts navigation */}

### Step 2: Locate the "More actions" Menu on the Draft Order Page

On the draft order detail page, find the **More actions** button in the top-right area. Click it to reveal the dropdown menu with Avada PDF Invoice action links.

{/* screenshot: Draft order detail page showing the "More actions" dropdown */}

### Step 3: Print a Draft Order Using "Print draft order by Avada"

Click **Print draft order by Avada** from the dropdown menu. The app opens a print preview page where you can select templates and a language, then preview and download the draft order invoice as a PDF.

{/* screenshot: Draft order print preview page with template selector and PDF preview */}

### Step 4: Download a Draft Order Using "Download draft order by Avada"

Click **Download draft order by Avada** from the dropdown menu. The app immediately generates and downloads the draft order invoice PDF to your computer using your default template.

{/* screenshot: Draft order download in progress */}

### Step 5: Select Multiple Draft Orders for Bulk Actions

Navigate back to the draft order list page (**Orders > Drafts**). Select multiple draft orders by clicking the checkboxes next to each one. A selection bar appears at the top of the list.

{/* screenshot: Draft order list with multiple orders selected */}

### Step 6: Mass Print Draft Orders Using "Print mass draft order by Avada"

With draft orders selected, click the **"..."** (more actions) button in the selection bar. Click **Print mass draft order by Avada**. The app opens a mass print preview page combining all selected draft order invoices into a single PDF.

On this page you can:
- **Select templates** from the checklist on the right sidebar
- **Choose a language** for the invoices
- **Preview** the combined PDF on the left

Click **Print** to download the combined PDF.

{/* screenshot: Mass print preview page for draft orders */}

### Step 7: Mass Download Draft Orders Using "Download mass draft order by Avada"

With draft orders selected, click the **"..."** button in the selection bar, then click **Download mass draft order by Avada**. The app generates and downloads a combined PDF containing all selected draft order invoices.

{/* screenshot: Mass download action in the selection bar dropdown */}

### Step 8: Customize the Print Output

On any print preview page (single or mass), use the right sidebar to:
- Select or deselect templates using the checkboxes
- Switch the invoice language using the language dropdown
- Click **View all templates** to go to the Templates page
- Click **Add new template** to create a new template

The PDF preview on the left updates automatically when you change selections.

{/* screenshot: Right sidebar with template and language options */}

### Step 9: Navigate Back to Draft Orders

Click the **Back** arrow button in the top-left of the print or download page to return to the previous page in Shopify Admin.

{/* screenshot: Back navigation button */}

## Tips

- Draft order extensions work the same way as regular order extensions, but use Draft Order-type templates.
- Make sure you have at least one template with the type set to "Draft Order" for the best results.
- The download action uses your default print template automatically, so set your preferred template as the default to save time.
- Unlike regular orders, draft orders do not have a "Send" extension since draft orders may not have a customer email associated yet.

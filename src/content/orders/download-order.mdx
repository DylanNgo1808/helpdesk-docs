# How to Download an Order as a PDF

Download individual order invoices as PDF files for your records, accounting, or to share manually with customers.

## Prerequisites

- Better PDF Invoice app installed on your Shopify store
- At least one order synced from your Shopify store
- The order must have a "Printable" status (visible in the **Print status** column)

## Steps

### Step 1: Open the Orders Page

Navigate to **Orders** in the left sidebar to view your order list.

{/* screenshot: Orders page with the order table visible */}

### Step 2: Locate the Order to Download

Find the order you want to download. Use the search bar or filters to narrow down the list if needed.

{/* screenshot: Order list with a specific order visible */}

### Step 3: Download from the Order List (Quick Action)

In the **Actions** column of the order row, click the **Download** icon (down arrow). The PDF file begins downloading immediately. A tooltip reading "Download" appears when you hover over the icon.

{/* screenshot: Download icon button in the Actions column with tooltip */}

### Step 4: Alternatively, Download from the Order Detail Page

Click on the order number to open the order detail page. Then click **Download** in the secondary actions bar at the top of the page.

{/* screenshot: Order detail page with Download button in secondary actions */}

### Step 5: Wait for the Download

A brief loading state appears while the PDF is generated. The file downloads to your browser's default download location with a filename based on your configured download file name pattern.

{/* screenshot: Loading state while PDF is being generated */}

### Step 6: Download via the Print Page

You can also download from the print preview page. Navigate to the print page for an order, then click the **Download** primary action button to save the PDF with your selected templates and language.

{/* screenshot: Download button on the print preview page */}

### Step 7: Verify the Download

Check your browser's download folder for the PDF file. The filename follows the pattern configured in your app settings, typically including the order number.

{/* screenshot: Downloaded PDF file in the browser downloads bar */}

## Tips

- Orders with an "Unprintable" print status require a plan upgrade to download. Hovering over the download icon shows "Upgrade now to unlock this feature."
- The download file name can be customized in the app settings.
- Downloading from the print page allows you to select specific templates and a language before generating the PDF.
- You can also use the Shopify admin "More actions" menu to trigger a download directly from within Shopify.

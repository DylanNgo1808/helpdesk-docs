# How to Filter Orders

The Orders page provides powerful filtering to help you narrow down your order list by status, payment, fulfillment, date, delivery method, tags, and customer type.

## Prerequisites

- Better PDF Invoice app installed on your Shopify store
- Orders synced from your Shopify store

## Steps

### Step 1: Open the Orders Page

Navigate to **Orders** in the left sidebar. The order list displays with filter tabs and a search bar at the top.

{/* screenshot: Orders page with filter tabs visible above the order table */}

### Step 2: Use Quick Filter Tabs

Click one of the preset tabs above the order table to quickly filter: **All**, **Opened**, **Archived**, **Cancelled**, **Unpaid**, or **Unfulfilled**. The selected tab highlights and the order list updates immediately.

{/* screenshot: Filter tabs showing All, Opened, Archived, Cancelled, Unpaid, Unfulfilled */}

### Step 3: Open Advanced Filters

Click the filter icon or the **Add filter** area next to the search bar to expand the advanced filter panel. This reveals all available filter categories.

{/* screenshot: Advanced filter panel expanded with filter category options */}

### Step 4: Filter by Payment Status

Select the **Payment status** filter and choose one or more values: **Paid**, **Partially paid**, **Refunded**, **Partially refunded**, **Payment pending**, **Overdue**, **Expired**, or **Unpaid**. The applied filter appears as a tag showing "Payment: [value]."

{/* screenshot: Payment status filter with multiple checkboxes and applied filter tag */}

### Step 5: Filter by Fulfillment Status

Select the **Fulfillment status** filter and choose from: **Fulfilled**, **Unfulfilled**, **Partially fulfilled**, or **On hold**. The applied filter tag shows "Fulfillment: [value]."

{/* screenshot: Fulfillment status filter with checkbox options */}

### Step 6: Filter by Date Range

Select the **Date** filter to choose a preset range: **Today**, **Last 7 days**, **Last 30 days**, **Last 90 days**, **Last 12 months**, or **Custom**. When selecting Custom, you can enter a specific **Starting** and **Ending** date using the date pickers.

{/* screenshot: Date filter with preset options and custom date picker fields */}

### Step 7: Filter by Type of Customer

Select the **Type of customer** filter and choose **Include** or **Exclude** to filter for B2B company orders or personal customer orders.

{/* screenshot: Type of customer filter with Include and Exclude options */}

### Step 8: Filter by Delivery Method

Select the **Delivery method** filter and choose from: **Local delivery**, **Local pickup**, or **Ship to customer**.

{/* screenshot: Delivery method filter with three options */}

### Step 9: Filter by Delivery Status

Select the **Delivery status** filter to narrow orders by shipping progress: **In transit**, **Out for delivery**, **Attempted delivery**, **Delayed**, **Failed delivery**, **Delivered**, **Tracking added**, or **No status**.

{/* screenshot: Delivery status filter showing all shipping status options */}

### Step 10: Filter by Tags

Use the **Tagged** filter to find orders with a specific tag, or use the **Not tagged** filter to exclude orders with a certain tag. Type the tag name in the text field.

{/* screenshot: Tagged and Not tagged filter text input fields */}

### Step 11: Filter by Discount Code

Enter a discount code in the **Discount code** filter to find orders that used a specific coupon or discount.

{/* screenshot: Discount code filter input field */}

### Step 12: Clear All Filters

Click **Clear all** to remove all active filters and return to the default order list view. Individual filter tags can also be removed by clicking the X on each applied filter tag.

{/* screenshot: Clear all button and applied filter tags with X buttons */}

## Tips

- Multiple filters can be combined. For example, filter by "Unpaid" payment status and "Unfulfilled" fulfillment status simultaneously.
- Applied filters persist when paginating through the order list.
- Changing any filter automatically refreshes the order list without needing to click a separate apply button.

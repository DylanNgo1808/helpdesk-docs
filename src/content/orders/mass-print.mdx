# How to Print Multiple Orders at Once

Bulk printing lets you generate a single PDF containing invoices for multiple selected orders. This is useful for printing batches of invoices for fulfillment or record-keeping.

## Prerequisites

- Better PDF Invoice app installed on your Shopify store
- Two or more orders synced from your Shopify store
- At least one invoice template configured with the template type "Order"

## Steps

### Step 1: Open the Orders Page

Navigate to **Orders** in the left sidebar to view your order list.

{/* screenshot: Orders page with order table visible */}

### Step 2: Select Multiple Orders

Check the checkboxes next to the orders you want to print. You can select orders individually or use the header checkbox to select all orders on the current page.

{/* screenshot: Multiple orders selected with checkboxes checked */}

### Step 3: Click Print Selected Orders

Once orders are selected, the bulk action bar appears. Click **Print selected orders** to navigate to the bulk print page.

{/* screenshot: Bulk action bar with Print selected orders button highlighted */}

### Step 4: Wait for the Preview

The page displays "Generating..." while the combined PDF is being created. Once ready, the preview appears in the main content area with the title "Print order."

{/* screenshot: Generating spinner on the mass print page */}

### Step 5: Review the Combined Preview

The left side shows an embedded PDF preview containing all selected orders' invoices combined into one document. Scroll through to verify all orders are included.

{/* screenshot: Combined PDF preview showing multiple invoices */}

### Step 6: Select Templates

On the right sidebar, under **Templates** with the description "Select templates to print your orders," choose which templates to apply to the invoices. Only Order-type templates appear in the list.

{/* screenshot: Template checkboxes on the right sidebar */}

### Step 7: Choose a Language

Use the **Choose language** dropdown to select the invoice language for all orders in the batch.

{/* screenshot: Language dropdown on the mass print page */}

### Step 8: Print or Download

Click the **Print** primary action button in the top-right corner to download the combined PDF file containing all selected orders.

{/* screenshot: Print button in the page header */}

### Step 9: Manage Templates

Click **View all** to navigate to the templates page, or click **Add template** to create a new template directly from this screen.

{/* screenshot: View all and Add template buttons below the template list */}

### Step 10: Navigate Back

Click the back arrow to return to the Orders page.

{/* screenshot: Back arrow in the top-left corner */}

## Tips

- If you select more orders than the maximum print limit, the app will send the combined PDF to your email as an attachment instead. A message confirms: "We started to send an email to your email with an attachment that contains all selected orders."
- On the Professional plan or higher, you can select orders across multiple pages using the **Total Selected** banner, then use **Print All Orders** to print them all at once.
- Templates marked as the default print template are auto-selected when the page loads.
- The preview updates automatically when you change the template or language selection.

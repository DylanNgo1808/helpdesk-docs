# How to Download Draft Orders as PDF

You can download draft order invoices as PDF files directly from the draft orders list, the draft order detail page, or the print page. You can also bulk download multiple draft orders at once.

## Prerequisites

- Better PDF Invoice app installed on your Shopify store
- A Professional or Ultimate plan (downloading requires a paid plan)
- At least one draft order in your store
- At least one draft order template configured

## Steps

### Step 1: Download from the draft orders list

On the **Draft Orders** page, find the draft order you want to download. Click the **Download** icon button (down arrow) in the Actions column for that row. The PDF file downloads automatically to your computer.

{/* screenshot: Download icon button in the Actions column of the draft orders table */}

### Step 2: Download from the draft order detail page

Open a draft order by clicking its name. On the detail page, click the **Download** button in the page header secondary actions. The app generates and downloads the PDF using your default template.

{/* screenshot: Download button in the draft order detail page header */}

### Step 3: Download from the print page

On the print preview page, click the **Download** primary action button at the top right. This downloads the PDF with the currently selected template and language settings.

{/* screenshot: Download button on the print preview page */}

### Step 4: Bulk download multiple draft orders

On the **Draft Orders** page, select multiple draft orders using the checkboxes. When the bulk action bar appears, click the **Download** button. The app generates a combined PDF and downloads it to your computer.

{/* screenshot: Bulk action bar with the Download button and selected order count */}

### Step 5: Wait for the download to complete

After initiating a download, a loading state appears. For single downloads, the file saves almost immediately. For bulk downloads of many orders, the download may take a few moments.

{/* screenshot: Loading state during PDF generation */}

### Step 6: Locate the downloaded file

The PDF file is saved to your browser's default download location. The file name uses the format configured in your settings (e.g., "YourStoreName_D1234.pdf"). For bulk downloads, all invoices are combined into a single PDF file.

{/* screenshot: Downloaded PDF file in the browser downloads bar */}

## Tips

- The download file name is based on your store name or the custom file name configured in your download settings. Spaces in the name are replaced with underscores.
- Single downloads use the default print template automatically. To download with a specific template, use the print page where you can select templates before downloading.
- Bulk downloads combine all selected draft orders into one PDF. If you need individual files, download each draft order separately.
- If you are on the Free plan, clicking the download button redirects you to the subscription page to upgrade.

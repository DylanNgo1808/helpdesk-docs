# How to Print a Draft Order Invoice

The print page lets you preview a draft order invoice as a PDF, choose a template, select a language, and download the file. You can access it from the draft orders list or the draft order detail page.

## Prerequisites

- Better PDF Invoice app installed on your Shopify store
- A Professional or Ultimate plan (printing requires a paid plan)
- At least one draft order template configured in your templates
- A desktop browser (print preview is not available on mobile devices)

## Steps

### Step 1: Navigate to the print page

There are two ways to reach the print page:

- **From the draft orders list**: Click the **Print** icon button in the Actions column for any draft order row.
- **From the draft order detail page**: Click the **Print** button in the page header actions.

{/* screenshot: Print icon button in the draft orders list Actions column */}

### Step 2: Wait for the preview to generate

After navigating to the print page, a loading spinner displays with the message "Generating PDF..." while the app fetches the draft order data and renders the PDF preview.

{/* screenshot: Loading spinner with "Generating PDF..." message */}

### Step 3: View the PDF preview

The left side of the page shows the full PDF preview in an A4-sized viewer. The preview automatically renders using your default print template for draft orders.

{/* screenshot: PDF preview panel showing the rendered draft order invoice */}

### Step 4: Select a template

On the right sidebar, the **Templates** card shows all available Draft Order templates as checkboxes. You can select one or multiple templates. The preview updates automatically when you change the selection. A description reads "Select templates for print."

{/* screenshot: Templates card with checkboxes for available Draft Order templates */}

### Step 5: Choose a language

Use the **Choose language** dropdown in the right sidebar to select the language for the invoice. This dropdown lists all supported languages. Changing the language requires a Professional plan or higher.

{/* screenshot: Language dropdown showing available language options with Pro badge */}

### Step 6: Download the PDF

Click the **Download** primary action button at the top right of the page to download the draft order invoice as a PDF file to your computer.

{/* screenshot: Download button in the page header */}

### Step 7: View the draft order in Shopify

Click the **View** button in the page header to open the original draft order in your Shopify admin in a new browser tab.

{/* screenshot: View button in the secondary actions of the page header */}

### Step 8: Manage templates

At the bottom of the templates card, click **View all templates** to go to the full templates management page, or click **Add template** to create a new draft order template (available on supported plans).

{/* screenshot: "View all templates" and "Add template" buttons at the bottom of the templates card */}

### Step 9: Return to draft orders

Click the **back arrow** at the top left of the page to return to the draft orders list.

{/* screenshot: Back arrow navigation in the page header */}

## Tips

- Selecting multiple templates generates a combined PDF with each template rendered as a separate section.
- The language dropdown is locked to your primary language on the free plan. Upgrade to Professional to unlock multi-language support.
- If the PDF preview does not display (e.g., on browsers that do not support embedded PDFs), the app automatically triggers a download instead.
- You can print directly from the browser by using your browser's native print function (Ctrl+P / Cmd+P) on the preview page.

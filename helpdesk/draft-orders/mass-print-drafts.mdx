# How to Print Multiple Draft Orders

The mass print feature lets you select multiple draft orders and generate a combined PDF containing all their invoices. This is useful when you need to print a batch of draft order invoices at once.

## Prerequisites

- Better PDF Invoice app installed on your Shopify store
- A Professional or Ultimate plan (bulk printing requires a paid plan)
- At least one draft order template configured
- Two or more draft orders in your store
- A desktop browser (mass print preview is not available on mobile devices)

## Steps

### Step 1: Open the Draft Orders page

In the Better PDF Invoice app, navigate to the **Draft Orders** page from the left navigation menu.

{/* screenshot: Draft Orders page with the navigation menu highlighted */}

### Step 2: Select draft orders using checkboxes

Click the checkboxes next to the draft orders you want to print. You can select individual rows or use the header checkbox to select all visible orders.

{/* screenshot: Draft orders table with multiple rows checked via checkboxes */}

### Step 3: Click the Print bulk action

After selecting draft orders, a bulk action bar appears at the top of the table. Click the **Print** button. If you are not on a Professional or higher plan, a "Professional" badge appears next to the button, and clicking it redirects you to the subscription page.

{/* screenshot: Bulk action bar showing the Print button with selected order count */}

### Step 4: Wait for the combined PDF to generate

The mass print page opens and displays a loading spinner while it generates the combined PDF for all selected draft orders.

{/* screenshot: Loading spinner on the mass print page */}

### Step 5: Preview the combined PDF

Once generated, the left side of the page shows the combined PDF preview containing all selected draft orders. Scroll through the preview to verify the invoices.

{/* screenshot: Combined PDF preview showing multiple draft order invoices */}

### Step 6: Select templates

On the right sidebar, the **Templates** card lists all available Draft Order templates as checkboxes. Select one or more templates to apply to the batch. The preview updates automatically.

{/* screenshot: Templates card with Draft Order template checkboxes */}

### Step 7: Choose a language

Use the **Choose language** dropdown to select the invoice language. The language applies to all draft orders in the batch. Multi-language support requires a Professional plan or higher.

{/* screenshot: Language dropdown in the right sidebar */}

### Step 8: Download the combined PDF

Click the **Print** primary action button at the top right of the page to download the combined PDF file containing all selected draft order invoices.

{/* screenshot: Print button in the page header for downloading the combined PDF */}

### Step 9: Return to draft orders

Click the **back arrow** in the page header to go back to the draft orders list.

{/* screenshot: Back arrow navigation at the top of the mass print page */}

## Tips

- There is a maximum number of draft orders you can print in a single batch. If you exceed the limit, the app will send the PDF to your email instead and display a message on screen.
- Selecting multiple templates generates each template for every draft order in the batch, so the final PDF may contain multiple pages per order.
- You can also bulk **Download** draft orders as separate PDFs from the draft orders list by selecting orders and clicking the **Download** bulk action button.
- The combined PDF uses the same file name format configured in your settings.

# How to View and Manage Company Orders

The company detail page provides a full order management experience for each B2B company. You can filter, sort, print, download, and combine orders directly from this view.

## Prerequisites

- A Wholesale plan in Better PDF Invoice
- B2B orders associated with companies in your Shopify admin
- B2B access scopes granted to the app

## Steps

### Step 1: Open a Company

Navigate to **B2B** in the sidebar, then click on a company name to open its detail page.

{/* screenshot: Company name clicked in the B2B company list */}

### Step 2: View the Order Table

The order index table displays all orders from the selected company. Each row shows:
- **Order name** (e.g., #1001) - links to the order detail view
- **Date** - when the order was created
- **Customer** - the customer who placed the order
- **Financial status** - payment status badge (e.g., Paid, Pending, Partially paid)
- **Fulfillment status** - shipping status badge (e.g., Fulfilled, Unfulfilled)
- **Total price** - the order total

{/* screenshot: Order index table with all columns visible */}

### Step 3: Filter Orders by Status

Use the status tabs at the top of the order table to filter by order status. You can filter by payment status (paid, unpaid) and fulfillment status (fulfilled, unfulfilled).

{/* screenshot: Status tabs above the order table */}

### Step 4: Search for Specific Orders

Type in the search field above the table to find orders by order number or customer name.

{/* screenshot: Search field with a search query */}

### Step 5: Sort Orders

Use the sort dropdown to reorder orders by different criteria such as date, order number, or total price.

{/* screenshot: Sort options dropdown */}

### Step 6: Apply Advanced Filters

Click on filter options to narrow results by financial status, fulfillment status, delivery method, delivery status, tags, discount codes, or date range.

{/* screenshot: Advanced filter panel with multiple filter options */}

### Step 7: Select Orders for Bulk Actions

Check the boxes next to orders to select them. The bulk action bar appears with the following options:
- **Print** - open selected orders for printing
- **Download** - download selected orders as PDF files
- **Download all** - download all filtered orders (requires Professional plan or higher)

{/* screenshot: Bulk action bar with Print, Download, and Download all buttons */}

### Step 8: Download a Single Order

For individual orders, click the download icon on the right side of the order row to download that specific order as a PDF.

{/* screenshot: Download icon on an order row */}

### Step 9: Print a Single Order

Click the print icon on an order row to open the print preview for that individual order.

{/* screenshot: Print icon on an order row */}

### Step 10: Navigate Between Order Pages

If there are more than 50 orders, use the pagination controls at the bottom of the table to browse additional pages.

{/* screenshot: Pagination controls at bottom of order table */}

## Tips

- You can combine multiple unpaid orders into a single invoice using the **Combine orders** bulk action. Select 2-10 unpaid orders to use this feature.
- Orders that are closed, paid, or missing payment terms cannot be combined.
- The **Download all** bulk action exports all filtered orders, not just the ones visible on the current page.
- Use the date filter to narrow results to specific time periods for easier management.

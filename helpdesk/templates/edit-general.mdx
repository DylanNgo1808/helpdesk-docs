# How to Edit Template General Settings

The **General** tab contains the foundational settings for your template: its name, currency display, order type, and document type. These settings determine how the template is categorized and which data it pulls from your Shopify orders.

## Prerequisites

- Better PDF Invoice app installed on your Shopify store
- An existing template, or a new template in the editor
- Access to the app admin dashboard

## Steps

### Step 1: Open the template editor

From the **Templates** page, click the **Edit** button on the template card you want to modify. The full-screen template editor opens.

{/* screenshot: Template card with the Edit button highlighted */}

### Step 2: Navigate to the General tab

In the left sidebar, click the **General** menu item. The help text reads "General settings for templates." The right panel updates to show the General settings form.

{/* screenshot: Left sidebar menu with General tab selected */}

### Step 3: Edit the template name

In the **Template name** text field, type a new name for your template. The name must be unique across all your templates. If you enter a name that already exists, a validation error message appears: "Template name already exists."

{/* screenshot: Template name field with sample text entered */}

### Step 4: Choose the template currency

Under **Template currency**, select one option:

- **Store currency** -- The currency displayed on the PDF matches your Shopify store's default currency. This is the default setting.
- **Billing currency** -- The currency displayed matches the billing currency from the transaction. This option requires the Ultimate plan. If you are not on the Ultimate plan, an **Ultimate** badge appears next to this option, and clicking it has no effect.

{/* screenshot: Template currency choice list with Store currency selected */}

### Step 5: Select the order type

Under **Order type**, two tabs are available:

- **Order** -- Standard Shopify orders (invoices, packing slips, refunds, etc.)
- **Draft order** -- Draft orders and quotes

Click a tab to switch between order types. Switching the order type reloads the template theme to match the appropriate layout.

{/* screenshot: Order type tabs showing "Order" and "Draft order" */}

### Step 6: Select the document type

From the **Document type** dropdown, select the specific document type for this template:

For **Order** type:
- Invoice
- Packing slip
- Refund
- Pro forma
- Credit note

For **Draft order** type:
- Quotes

Changing the document type reloads the template theme to use the correct layout for that document type.

{/* screenshot: Document type dropdown expanded showing all options */}

### Step 7: Review the live preview

After making changes, check the center preview panel. The PDF preview automatically refreshes to reflect your updated settings, including any layout changes triggered by switching the document type.

{/* screenshot: Live preview panel showing the updated template */}

### Step 8: Save your changes

Click **Save** in the save bar at the bottom of the editor to apply your changes. A confirmation toast appears when the save is successful.

{/* screenshot: Save bar with the Save button highlighted */}

## Tips

- Template names must be unique -- use descriptive names like "Standard Invoice" or "Holiday Packing Slip" to distinguish between templates
- Billing currency is a premium feature. If you need it, upgrade to the Ultimate plan from the Subscription page
- Changing the order type or document type reloads the template layout, so make sure to review the preview after switching
- Use the **Discard** button if you want to revert all changes since your last save

# How to Browse and Filter Templates

The **Templates** page displays all your PDF document templates in a visual grid. You can filter by document type, preview template designs, and quickly identify which template is set as your default.

## Prerequisites

- Better PDF Invoice app installed on your Shopify store
- Access to the app admin dashboard

## Steps

### Step 1: Open the Templates page

From the Better PDF Invoice sidebar menu, click **Templates**. The page loads with all your templates displayed as thumbnail cards in a grid layout.

{/* screenshot: Templates page showing the grid of template cards */}

### Step 2: Filter templates by document type

At the top of the page, use the tab bar to filter templates by type. Available tabs include:

- **Invoice** -- Standard order invoices
- **Packing slip** -- Shipment packing slips
- **Refund** -- Refund receipt templates
- **Pro forma** -- Pro forma invoices
- **Credit note** -- Credit note documents
- **Quote** -- Quote templates for draft orders

Click a tab to show only templates of that document type.

{/* screenshot: Tab bar showing document type filters (Invoice, Packing slip, Refund, etc.) */}

### Step 3: Review template cards

Each template card displays:

- A **thumbnail preview** of the template design
- The **template name** in bold text
- A **badge** indicating whether the template is set as **Print by default** or **No default**
- A **template type badge** (e.g., Order, Draft Order)

{/* screenshot: A single template card showing thumbnail, name, and badges */}

### Step 4: Identify your default template

Look for the green **Print by default** badge on a template card. This template is automatically used when printing, sending, or downloading invoices. Templates without this designation show a **No default** badge.

{/* screenshot: Template card with the green "Print by default" badge highlighted */}

### Step 5: Select or unselect a template as default

Click the **Select** button on a template card to set it as the default for its document type. If a template is already selected, the button reads **Unselect** -- click it to remove the default designation.

{/* screenshot: Template card with the Select/Unselect button highlighted */}

### Step 6: Preview a template

Click the **Preview** button on any template card to open a larger preview in a modal window. The modal shows the template name as the title and a full-size thumbnail of the template layout.

{/* screenshot: Template preview modal showing the full-size template design */}

### Step 7: View premium templates

Templates from higher plans (Professional, Ultimate) appear with a colored **Professional** or **Ultimate** badge. Click **Upgrade** to access these templates, or click **Preview** to see the design before upgrading.

{/* screenshot: Premium template card with the "Ultimate" badge and Upgrade button */}

## Tips

- Only one template per document type can be set as the default for printing and sending
- Use the **Preview** button to compare template designs before selecting one
- Premium templates require a plan upgrade -- contact support if you want to unlock a single premium template for free

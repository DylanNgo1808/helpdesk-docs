# How to Change Language and Locale Settings

Better PDF Invoice supports multiple languages for your invoice documents and emails. This guide shows you how to set your primary language, translate invoice labels, and enable advanced language features.

## Prerequisites

- Better PDF Invoice installed on your Shopify store
- A Professional plan or higher is required for changing the primary invoice language (free plans use English)

## Steps

### Step 1: Open Settings

From the Better PDF Invoice app, click **Settings** in the app navigation to open the Settings page.

{/* screenshot: Better PDF Invoice navigation with "Settings" highlighted */}

### Step 2: Navigate to Translation

In the left sidebar, click **Translation**. This section is identified by a translate icon and contains all language-related settings.

{/* screenshot: Settings sidebar with "Translation" highlighted */}

### Step 3: Set Your Primary Language

In the **Advanced settings** section at the top, locate the **Primary language** dropdown. Select your preferred language from the list of available languages. This sets the default language used for all invoice labels and document text.

{/* screenshot: Advanced settings section showing the "Primary language" dropdown with language options */}

### Step 4: Enable Language at Billing Address (Optional)

Below the primary language, you will see three advanced language features:

- **Send PDF document with language at billing address** (Ultimate plan) -- Automatically sends PDF documents using the language detected from the customer's billing address.
- **Use email language at billing address** (Professional plan) -- Uses the language detected from the billing address for emails.
- **Download PDF document with language at billing address** (Ultimate plan) -- Downloads PDFs in the language of the customer's billing address.

Click **Turn on** next to any feature to enable it, or **Turn off** to disable it.

{/* screenshot: Advanced settings showing the three language feature rows with Turn on/Turn off buttons and plan badges */}

### Step 5: Choose a Language to Translate

Below the advanced settings, in the **Translation** card, use the **Choose language to translate to** dropdown to select the target language you want to edit translations for.

{/* screenshot: The Translation card with the "Choose language to translate to" dropdown */}

### Step 6: Select a Translation Tab

Translation labels are organized into tabs by document type. Click on the tab you want to edit:

- **General** -- Common labels like address, phone, tax ID, payment method, shipping method
- **Invoice** -- Invoice-specific labels like invoice date, invoice amount
- **Quote** -- Quote/draft order labels
- **Packing slip** -- Packing slip labels like pack date, pack amount
- **Refund** -- Refund-specific labels like refund date, refund amount
- **Order** -- Order labels like order date, order amount
- **Email** -- Email subject lines, greetings, and signature text

{/* screenshot: Translation tabs showing General, Invoice, Quote, Packing slip, Refund, Order, and Email tabs */}

### Step 7: Search for a Specific Label

Use the **Search translation** field at the top of the labels list to quickly find a specific label by name. Type any part of the label name to filter the results.

{/* screenshot: The search field with "Search translation" placeholder and the search icon */}

### Step 8: Edit Translation Labels

Each translation label is displayed as a text field with the label name and current value. Edit any field to change the translated text. For example, change "Subtotal" to your preferred translation in the selected language.

{/* screenshot: Translation editor showing editable text fields for labels like "Subtotal", "Tax", "Grand total" */}

### Step 9: Save Your Translations

After editing your translations, click the **Save** button in the save bar at the top of the page. A "Save successfully" message will confirm your changes.

{/* screenshot: The save bar at the top with the "Save" button and success toast */}

## Tips

- The primary language setting affects all new documents. Previously generated invoices retain their original language.
- If you are on a free plan, the primary language is set to English. Upgrade to a Professional plan to unlock other languages.
- The "Send PDF document with language at billing address" feature is powerful for stores selling internationally -- it automatically sends invoices in the customer's local language.
- Use the search field to quickly locate specific labels instead of scrolling through the full list.
- If you have B2B features enabled, an additional **B2B** tab will appear with labels for statements, due dates, and payment status.

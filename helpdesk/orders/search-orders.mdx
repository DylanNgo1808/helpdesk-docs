# How to Search for Specific Orders

Use the search bar on the Orders page to quickly find a specific order by its order number or the customer's email address.

## Prerequisites

- Better PDF Invoice app installed on your Shopify store
- Orders synced from your Shopify store

## Steps

### Step 1: Open the Orders Page

Navigate to **Orders** in the left sidebar to view your order list.

{/* screenshot: Orders page with the search bar visible at the top */}

### Step 2: Locate the Search Bar

At the top of the order table, find the search field with the placeholder text "Search by Order Number or Customer Email."

{/* screenshot: Search bar with placeholder text visible */}

### Step 3: Search by Order Number

Type the order number (e.g., `1001`) into the search field. The order list updates automatically after a brief delay as you type. You do not need to include the # symbol.

{/* screenshot: Search bar with order number entered and filtered results */}

### Step 4: Search by Customer Email

Type the customer's email address (e.g., `johndoe@example.com`) into the search field. The order list filters to show all orders associated with that email.

{/* screenshot: Search bar with email entered and matching orders displayed */}

### Step 5: Review Search Results

The order table updates to show only orders matching your search query. All table columns and features remain available, including sorting, selecting, and quick actions.

{/* screenshot: Filtered order list showing search results with full table columns */}

### Step 6: Clear the Search

Click the **X** button inside the search field to clear your search query and return to the full order list. Alternatively, manually delete the text in the search field.

{/* screenshot: Search bar with X clear button highlighted */}

### Step 7: Combine Search with Filters

You can use the search bar together with any active filters. For example, search for an order number while also filtering by "Unpaid" payment status to find a specific unpaid order.

{/* screenshot: Search combined with an active payment status filter */}

## Tips

- The search executes automatically after you stop typing, with a short delay to avoid unnecessary requests.
- Enter the order name without the # prefix for best results (e.g., type `1001` instead of `#1001`).
- The help text below the search bar reminds you: "You should enter the order name to search (eg: 1001), you can also search by customer email (eg: johndoe@example.com)."

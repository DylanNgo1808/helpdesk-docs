# How to View and Navigate Your Order List

The Orders page displays all orders received via Shopify. From here you can browse, sort, select, and take actions on your invoices.

## Prerequisites

- Better PDF Invoice app installed on your Shopify store
- At least one order placed in your Shopify store

## Steps

### Step 1: Open the Orders Page

In the Better PDF Invoice app, click **Orders** in the left sidebar navigation. The page title reads "Orders" with the subtitle "Orders received via Shopify will be shown here."

{/* screenshot: Orders page showing the full order list with title and subtitle */}

### Step 2: Review the Order Table Columns

Each order row displays the following columns: **Orders** (order number), **Date**, **Customer**, **Type of customer**, **Total**, **Payment status**, **Fulfillment status**, **Item** count, **Delivery method**, **Delivery status**, **Tag**, **Print status**, and **Actions**.

{/* screenshot: Order table headings showing all column names */}

### Step 3: Understand Order Badges

Each order shows a **Payment status** badge (e.g., Paid, Unpaid, Partially paid, Refunded) and a **Fulfillment status** badge (e.g., Fulfilled, Unfulfilled, Partially fulfilled). The **Print status** column displays either "Printable" or "Unprintable."

{/* screenshot: Order row with payment status and fulfillment status badges */}

### Step 4: View Customer Information

Click a customer name in the **Customer** column to open a popover with the customer's contact details, email, and related information. Customers without an associated record display as "No customer."

{/* screenshot: Customer popover showing contact details and email */}

### Step 5: View Order Notes

Orders with notes display a note icon next to the order number. Hover over the icon to see the note content in a tooltip.

{/* screenshot: Note icon tooltip showing the order note */}

### Step 6: View Order Tags

The **Tag** column shows up to two tags per order. If there are more than two, a "+N" indicator appears showing the remaining count.

{/* screenshot: Order row with tags and overflow count */}

### Step 7: Sort the Order List

Use the sort controls above the order table to sort by **Orders** (Oldest/Newest) or **Price** (Low to high/High to low). Click the sort direction to toggle between ascending and descending.

{/* screenshot: Sort dropdown with Orders and Price sort options */}

### Step 8: Navigate Between Pages

Use the **Pagination** controls at the bottom of the order table to move between pages. Click the left arrow to go to the previous page and the right arrow to go to the next page.

{/* screenshot: Pagination controls at the bottom of the order list */}

### Step 9: Select Orders for Bulk Actions

Check the checkbox next to individual orders to select them, or use the header checkbox to select all orders on the current page. Selected orders reveal bulk action buttons: **Print selected orders**, **Download selected orders**, and **Download all orders**.

{/* screenshot: Selected orders with bulk action buttons visible */}

### Step 10: Use Quick Actions on Individual Orders

Each order row has quick-action buttons in the **Actions** column: a **Print** icon to print the invoice and a **Download** icon to download the PDF. Hover over the icons to see the tooltip labels "Print" and "Download."

{/* screenshot: Actions column with print and download icon buttons */}

### Step 11: Open an Order Detail

Click on an order number link (e.g., #1001) to navigate to the order detail page where you can view the full order information, send invoices, and manage the order.

{/* screenshot: Clicking an order number to open the order detail view */}

## Tips

- Orders from canceled orders are shown with a strikethrough on the order number, customer name, and total.
- The **Total Selected** banner appears when selecting orders across multiple pages, letting you manage cross-page selections.
- Use the **Clear** button in the Total Selected banner to deselect all orders across pages.

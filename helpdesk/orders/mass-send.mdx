# How to Send Invoices to Multiple Customers

Bulk send allows you to email invoices to multiple customers simultaneously. This is triggered from the Shopify admin through the "More actions" menu on the orders page.

## Prerequisites

- Better PDF Invoice app installed on your Shopify store
- Multiple orders with valid customer email addresses
- Orders must have a "Printable" status

## Steps

### Step 1: Open Your Shopify Admin Orders Page

In your Shopify admin, navigate to **Orders** to view your order list.

{/* screenshot: Shopify admin orders page */}

### Step 2: Select Multiple Orders

Check the checkboxes next to the orders you want to send invoices for.

{/* screenshot: Multiple orders selected in Shopify admin */}

### Step 3: Open the More Actions Menu

Click the **More actions** button (or the three-dot menu) that appears when orders are selected.

{/* screenshot: More actions menu in Shopify admin with orders selected */}

### Step 4: Select the Send Invoice Option

From the dropdown menu, select the Better PDF Invoice send option. This opens the app's bulk send process.

{/* screenshot: Send invoice option in the More actions dropdown */}

### Step 5: Wait for Processing

A "Generating..." loading screen appears while the app processes and sends each invoice to the respective customer emails.

{/* screenshot: Generating spinner during bulk send processing */}

### Step 6: Confirm Successful Send

Once all invoices are sent, a success toast message confirms: "Sent successfully." The app redirects you back to the main page.

{/* screenshot: Success toast message after bulk send completes */}

### Step 7: Review Any Errors

If any invoices fail to send (e.g., due to missing email addresses or invalid data), an error banner appears listing the issues. Review the errors and address them for the affected orders individually.

{/* screenshot: Error banner showing failed send details */}

## Tips

- Each invoice is sent to the email address on file for that order's customer. Ensure customer emails are correct in Shopify before sending.
- Invoices are sent using your default invoice template and language settings.
- If an order does not have a customer email, the send will fail for that specific order.
- You can also send individual invoices from the order detail page using the **Send invoice** button for more control over the template and language.

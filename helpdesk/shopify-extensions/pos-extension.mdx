# How to Use Better PDF Invoice with Shopify POS

Better PDF Invoice includes a Point of Sale (POS) extension that adds a **Print Invoice** action to the order details screen in the Shopify POS app. You can select templates, choose a language, preview the invoice, and print it directly from your POS device.

## Prerequisites

- Better PDF Invoice app installed and active on your Shopify store
- Shopify POS app installed on your device (iPad, iPhone, or Android)
- An active subscription plan that supports POS printing
- At least one completed POS order

## Steps

### Step 1: Complete a Sale in Shopify POS

Process an order as you normally would in the Shopify POS app. After completing the transaction, navigate to the order details screen.

{/* screenshot: Shopify POS completed sale screen */}

### Step 2: Open the Order Details Screen

From the Shopify POS app, tap on the order you want to print an invoice for. This opens the order details screen showing the order summary, line items, and customer information.

{/* screenshot: POS order details screen showing order summary */}

### Step 3: Tap the "More actions" Menu

On the order details screen, tap the **"..."** (more actions) menu button. This reveals additional action items including the Better PDF Invoice print option labeled **pdf-pos**.

{/* screenshot: POS order details "more actions" menu with pdf-pos option */}

### Step 4: Open the Print Invoice Modal

Tap the **pdf-pos** action from the menu. A modal screen titled **Print Invoice** appears, providing template selection, language options, and a live print preview.

{/* screenshot: Print Invoice modal opened on the POS device */}

### Step 5: Choose a Language

If your plan supports language selection (Pro plan or higher), you will see a **Choose language** section at the top of the modal. Tap the language button to expand a list of all available languages. Tap on a language to select it.

If your plan does not support language selection, an informational banner displays: "Language selection requires Pro plan."

{/* screenshot: Language selection list expanded on the POS Print Invoice modal */}

### Step 6: Select Templates

Below the language section, you will see checkboxes for each of your available invoice templates. Tap to select or deselect templates. You can select multiple templates to include them all in the printed output.

The default print template is pre-selected automatically. If no default is set, the first template is selected.

{/* screenshot: Template checkboxes on the POS modal with one template selected */}

### Step 7: Preview the Invoice

Below the template selection area, a live print preview renders the invoice using your selected template(s) and language. Scroll to review the invoice content before printing.

{/* screenshot: Live PDF print preview on the POS modal */}

### Step 8: Print the Invoice

Tap the **Print** button below the template list. The POS app sends the invoice to your connected printer. The Print button is disabled if no templates are selected.

{/* screenshot: Print button on the POS modal */}

### Step 9: Close the Modal

After printing, tap outside the modal or use the close/back gesture to dismiss the Print Invoice screen and return to the order details.

{/* screenshot: Returning to order details after printing */}

## Tips

- Make sure your POS device is connected to a receipt printer or compatible printer before attempting to print.
- Set your most-used template as the **Default for printing** in the app's template settings to save time at the register.
- If the modal shows an "Upgrade required to print invoices" banner, your plan does not support POS printing. Visit the Subscription page in the app to upgrade.
- The POS extension renders the same invoice templates used in the web admin, so any customizations you make to your templates will also appear on POS printouts.
- Language selection on POS follows the same plan restrictions as the web app.
